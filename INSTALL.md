# Requirements

## Arch

```
yay -S libX11 libxft libxinerama libxrandr xinit
```

## fedora

```bash
dnf install libX11-devel libXft-devel libXinerama-devel  libXrandr-devel xorg-x11-xinit-session
```

# Install the dwm

```bash
cd dwm
./install
```

# Install the dmenu

```bash
cd dmenu
./install
```

# Install the st terminal

```bash
cd st
./install
```
